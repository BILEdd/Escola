# 🏫 Sistema Escolar

Sistema desenvolvido em Java com foco em Programação Orientada a Objetos (POO), 
permitindo o cadastro e gerenciamento de alunos e professores.

## 🛠️ Tecnologias Utilizadas

- Java
- Programação Orientada a Objetos (POO)

## 📋 Funcionalidades

- Cadastro de alunos
- Cadastro de professores
- Modelagem de entidades com classes e objetos

## 🚀 Como executar

1. Clone o repositório
   git clone https://github.com/BILEdd/Escola.git

2. Compile os arquivos Java
   javac *.java

3. Execute o programa
   java exe

## 📚 Aprendizados

Projeto desenvolvido para praticar os conceitos de POO como 
encapsulamento, classes, objetos e atributos em Java.
